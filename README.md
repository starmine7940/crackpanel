# 概要
オンライン会をする際に、ブラウザで手軽に遊べるゲームがあったら盛り上がると思い、作成しました。ルールのシンプルさと作戦の複雑性のバランスが特徴です。データベースはFirestoreを使うことで、リアルタイムの通信を可能にしました。また、SPAで作成しているため、サクサク動作します。

使った技術: HTML, CSS, JavaScript, Firebase
